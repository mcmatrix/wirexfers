Supported Payment Providers
===========================

List of currently supported protocols:

 * IPizza
 * Solo/TUPAS

Providers supporting IPizza protocol
------------------------------------

* LHV Bank Estonia
* Sampo Bank Estonia
* SEB Bank Estonia
* Swedbank Estonia

----------------------------------------
